#### <span id="drill-module">设备payload定义</span>
!> 设备payload使用json格式传输 


##### payload定义


> ###### PM2.5传感器 0301

```json
{
    level:"1",  //1-5
    desc:"空气优",
    value:"20",
    history:[ //返回7天
        {
            date:"10-20",
            value:"50"
        },
        {
            date:"10-21",
            value:"42"
        }
    ]
} 
```
