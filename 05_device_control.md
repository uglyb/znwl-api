#### 设备控制命令
通过该接口控制设备
`客户端相关设备通道定义，与后台一致`

**接口地址:**  `api/v1/device/control/command`

**请求参数**

|      参数     | 是否可选 | 参数说明 |
|---------------|----------|----------|
| deviceId      | 否       | 设备 ID  |
| deviceMode    | 否       | 设备型号 |
| channelNumber | 否       | 设备通道 |
| channelValue  | 否       | 通道值   |



** 响应参数 **

`无`

** 响应示例 **

`无`
